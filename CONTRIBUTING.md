# Contributing to the Agent Engineer course

Thanks for your interest in improving this course. Here is how you can help.

## Ways to contribute

If you're going to use AI to contribute, please follow the guidelines above and be transparent about it in your PR description. Here are some specific ways you can contribute:

- **Fix typos and errors** - Found a mistake? Open a PR with the fix.
- **Improve explanations** - If something is confusing, suggest a clearer way to explain it.
- **Add links** - Know a great codelab, tutorial, or doc page that is relevant? Add it.
- **Report issues** - Something outdated or broken? Open an issue.

## Guidelines

- Keep the tone conversational and accessible. This course is for beginners.
- Use analogies where they help. Avoid jargon without explanation.
- Use "-" instead of emdash characters.
- Use simple characters only - no special Unicode or emoji.
- Link to official docs rather than duplicating API references or code samples.
- Keep PRs focused. One change per PR is easier to review.

## What we are not looking for

- Full code samples or tutorials. We link to official resources for that.
- Content that duplicates existing Google Cloud documentation.
- Marketing language or promotional content.

## Process

1. Fork the repo
2. Create a branch for your change
3. Make your edits
4. Open a PR with a clear description of what you changed and why

## Code of conduct

Be kind, be respectful, be helpful. We are all here to learn.
